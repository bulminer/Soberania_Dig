<h1>Deixa o Google Docs e Usa o OnlyOffice ou CryptPad</h1>
<p>O <strong>Google Docs</strong> é útil para edição colaborativa, mas <strong>todos os teus documentos estão armazenados nos servidores do Google</strong>, onde podem ser analisados. Felizmente, a Europa tem alternativas soberanas: <a href="https://www.onlyoffice.com/">OnlyOffice</a> (França) e <a href="https://cryptpad.fr/">CryptPad</a> (França).</p>

<h2>Porque Deves Migrar?</h2>
<ul>
  <li><strong>Privacidade:</strong> OnlyOffice e CryptPad não analisam os teus documentos.</li>
  <li><strong>Sede na UE:</strong> Cumprem o GDPR, ao contrário do Google.</li>
  <li><strong>Open-source:</strong> Ambos são transparentes e auditáveis.</li>
  <li><strong>Encriptação:</strong> CryptPad encripta documentos de ponta a ponta.</li>
</ul>

<h2>Como Migrar do Google Docs</h2>
<ol>
  <li><strong>Experimenta o OnlyOffice:</strong>
    <ul>
      <li>Acede a <a href="https://www.onlyoffice.com/">onlyoffice.com</a> e cria uma conta.</li>
      <li>Faz upload de documentos do Google Drive para o OnlyOffice.</li>
    </ul>
  </li>
  <li><strong>Usa o CryptPad para documentos sensíveis:</strong>
    <ul>
      <li>Acede a <a href="https://cryptpad.fr/">cryptpad.fr</a> (não precisa de registo).</li>
      <li>Cria um novo documento e partilha o link com colegas (os documentos são encriptados).</li>
    </ul>
  </li>
  <li><strong>Integra com Nextcloud (opcional):</strong>
    <ul>
      <li>Se usas Nextcloud, podes instalar o <a href="https://nextcloud.com/collabora/">Collabora Online</a> para edição de documentos.</li>
    </ul>
  </li>
  <li><strong>Exporta os teus documentos do Google Docs:</strong>
    <ul>
      <li>No Google Drive, seleciona os ficheiros > Faz download (em formato .docx ou .odt).</li>
      <li>Faz upload para o OnlyOffice ou CryptPad.</li>
    </ul>
  </li>
</ol>
