<h1>Deixa o Google Chrome e Usa o Brave ou Firefox</h1>
<p>O <strong>Google Chrome</strong> é o browser mais usado do mundo, mas <strong>envia dados sobre os teus hábitos de navegação para o Google</strong>, que os usa para anúncios personalizados. Felizmente, existem alternativas focadas na privacidade: <a href="https://brave.com/">Brave</a> (EUA, mas com políticas de privacidade fortes) e <a href="https://www.mozilla.org/firefox/">Firefox</a> (EUA, mas open-source e com sede na Europa).</p>

<h2>Porque Deves Migrar?</h2>
<ul>
  <li><strong>Privacidade:</strong> Brave bloqueia rastreio e anúncios por defeito. Firefox tem proteções avançadas contra rastreio.</li>
  <li><strong>Sem dados para o Google:</strong> Chrome envia dados de utilização para o Google; Brave e Firefox não.</li>
  <li><strong>Open-source:</strong> Firefox é totalmente open-source e auditável.</li>
  <li><strong>Velocidade:</strong> Brave é mais rápido que o Chrome porque bloqueia anúncios.</li>
</ul>

<h2>Como Migrar do Google Chrome</h2>
<ol>
  <li><strong>Descobre o Brave:</strong>
    <ul>
      <li>Descarrega o Brave para <a href="https://brave.com/download/">Windows</a>, <a href="https://brave.com/download/">Mac</a> ou <a href="https://play.google.com/store/apps/details?id=com.brave.browser">Android</a>/<a href="https://apps.apple.com/app/brave-private-web-browser/id1052879175">iOS</a>.</li>
    </ul>
  </li>
  <li><strong>Importa os teus dados:</strong>
    <ul>
      <li>Ao abrir o Brave pela primeira vez, seleciona "Importar dados do Chrome" (favoritos, histórico, passwords).</li>
    </ul>
  </li>
  <li><strong>Define o Brave como browser padrão:</strong>
    <ul>
      <li>No Brave, vai a Definições > Definir como padrão.</li>
    </ul>
  </li>
  <li><strong>Experimenta o Firefox:</strong>
    <ul>
      <li>Descarrega o Firefox em <a href="https://www.mozilla.org/firefox/">mozilla.org/firefox</a>.</li>
      <li>Ativa as proteções de privacidade: Vai a Definições > Privacidade e Segurança > Seleciona "Estrito".</li>
    </ul>
  </li>
  <li><strong>Apaga os dados do Chrome:</strong>
    <ul>
      <li>No Chrome, vai a Definições > Privacidade e segurança > Limpar dados de navegação.</li>
    </ul>
  </li>
</ol>

<h2>Dicas para Navegação Privada</h2>
<ul>
  <li>No Brave, ativa o <a href="https://brave.com/privacy-features/">Brave Shields</a> para bloquear rastreio.</li>
  <li>No Firefox, usa extensões como <a href="https://addons.mozilla.org/firefox/addon/https-everywhere/">HTTPS Everywhere</a> e <a href="https://addons.mozilla.org/firefox/addon/privacy-badger17/">Privacy Badger</a>.</li>
  <li>Experimenta o <a href="https://librewolf.net/">LibreWolf</a> para uma versão do Firefox ainda mais privada.</li>
</ul>

<h2>Conclusão</h2>
<p>O Google Chrome é rápido, mas <strong>compromete a tua privacidade</strong>. Brave e Firefox oferecem a mesma velocidade e funcionalidades, mas sem rastreio. <strong>Muda hoje e navega com mais segurança!</strong></p>
